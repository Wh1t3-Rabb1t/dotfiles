# Visual Bindings


## MODE TOGGLING -------------------------------------------

- `Esc`                  : Command mode
- `s`                    : Command mode
- `S`                    : Visual line mode


## LINE NAVIGATION -----------------------------------------

- `o`                    : Forward word
- `u`                    : Backward word
- `O`                    : Forward word end
- `U`                    : Backward word end
- `a`                    : Exchange point and mark


## DELETE BINDINGS -----------------------------------------

- `BS`                   : Kill region
- `Del`                  : Kill region
- `w`                    : Kill region


## SELECT IN WORD / LINE -----------------------------------

- `Alt Right`            : Visual line mode


## UPPER / LOWER / SWAP CASE -------------------------------

- `-`                    : Up case
- `_`                    : Down case


## COPY ----------------------------------------------------

- `c`                    : Copy to clipboard


## CUT -----------------------------------------------------

- `x`                    : Cut to clipboard


## CHANGE --------------------------------------------------

- `y`                    : Change


## PASTE ---------------------------------------------------

- `v`                    : Paste from clipboard


## ADD SURROUNDING -----------------------------------------

Any quote or opening bracket character will add a pair of
said character to either side of the visual selection.
