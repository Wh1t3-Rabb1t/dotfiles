# Viins Bindings


## MODE TOGGLING -------------------------------------------

- `Esc`       : Command mode


## LINE NAVIGATION -----------------------------------------

- `Home`      : Beginning of line
- `End`       : End of line
- `Alt Left`  : Backward word
- `Alt Right` : Jump forward word


## DELETE BINDINGS -----------------------------------------

- `BS`        : Backward delete char
- `Del`       : Delete char
- `Alt BS`    : Backward kill word
- `Alt Del`   : Kill word
- `Ctrl BS`   : Backward kill line
- `Ctrl Del`  : Kill line
- `Shift Del` : Kill whole line


## UNDO / REDO ---------------------------------------------

- `Alt y`     : Undo
- `Alt Y`     : Redo


## PASTE ---------------------------------------------------

- `Alt v`     : Paste from clipboard
