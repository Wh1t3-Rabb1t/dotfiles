# Vi Insert Bindings


## MODE TOGGLING ---------------------------------------------------------------
`Esc` : Vi-insert
`s` : Visual-mode
`S` : Visual-line-mode


## LINE NAVIGATION -------------------------------------------------------------

`Home` : Beginning-of-line
`End` : End-of-line
`Alt Left` : Vi-backward-word
`Alt Right` : Jump_forward_word


## DELETE BINDINGS -------------------------------------------------------------

`BS` : Backward-delete-char
`Del` : Delete-char
`Alt BS` : Backward-kill-word
`Alt Del` : Kill-word
`Ctrl BS` : Backward-kill-line
`Ctrl Del` : Kill-line
`Shift Del` : Kill-whole-line


## UNDO / REDO -----------------------------------------------------------------

- `Alt y` : Undo
- `Alt Y` : Redo


## PASTE -----------------------------------------------------------------------

- `Alt v` : Paste_from_clipboard
